Deskripsi

Studi ini bertujuan untuk membandingkan performa dua algoritma pengurutan sederhana, yaitu Bubble Sort dan Insertion Sort, dalam mengurutkan data rating makanan dari restoran pada platform GoFood. Pengujian ini untuk mengevaluasi efisiensi setiap algoritma berdasarkan waktu yang dibutuhkan (running time).

Analisis akan menunjukkan bagaimana teori kompleksitas waktu O(nÂ²) diterapkan dalam praktik dan algoritma mana yang lebih cepat dalam situasi tertentu. Dengan membandingkan kedua algoritma, studi ini memberikan wawasan tentang kelebihan dan kekurangan masing-masing serta situasi mana yang lebih cocok untuk digunakan dalam pengurutan rating makanan.

Studi Kasus

Kami akan melakukan perbandingan antara dua algoritma pengurutan yaitu Bubble Sort sebagai algoritma iteratif dan Insertion Sort sebagai algoritma rekursif dalam konteks pengurutan data rating makanan dari restoran yang ada di Yogyakarta pada platform Gofood.

Implementasi Algoritma

Bubble Sort Implementation

Implementasi Bubble Sort menggunakan pendekatan iteratif untuk mengurutkan data rating GoFood secara descending.


// BubbleSort untuk mengurutkan berdasarkan rating
func BubbleSort(data [][]string) [][]string {
    n := len(data)
    for i := 0; i < n-1; i++ {
        for j := 0; j < n-i-1; j++ {
            rating1, _ := strconv.ParseFloat(data[j][0], 64)
            rating2, _ := strconv.ParseFloat(data[j+1][0], 64)
            if rating1 < rating2 { // Tukar jika rating1 lebih kecil
                data[j], data[j+1] = data[j+1], data[j]
            }
        }
    }
    return data
}

Recursive Insertion Sort Implementation

Implementasi Insertion Sort menggunakan pendekatan rekursif untuk mengurutkan data rating GoFood secara descending.


// Rekursif insertion sort helper untuk menyisipkan elemen ke posisi yang tepat
func insert(data [][]string, i int) {
    if i <= 0 {
        return
    }
    key := data[i]
    j := i - 1
    for j >= 0 {
        rating1, _ := strconv.ParseFloat(data[j][0], 64)
        ratingKey, _ := strconv.ParseFloat(key[0], 64)
        if rating1 < ratingKey {
            data[j+1] = data[j]
            j--
        } else {
            break
        }
    }
    data[j+1] = key
}

// Rekursif insertion sort untuk mengurutkan data
func RecursiveInsertionSort(data [][]string, i int) [][]string {
    if i == len(data) { // Basis kasus
        return data
    }
    insert(data, i)                          // Sisipkan elemen ke tempat yang sesuai
    return RecursiveInsertionSort(data, i+1) // Rekursi ke elemen berikutnya
}

Comparison Analysis Implementation

Implementasi kode untuk membandingkan waktu eksekusi kedua algoritma dengan berbagai ukuran data.


func main() {
    // Ukuran data yang akan diuji
    sizes := []int{10, 50, 100, 200, 500, 800, 1000}
    var bubbleTimes []float64
    var insertionTimes []float64

    for _, size := range sizes {
        sampleData := make([][]string, size)
        copy(sampleData, data[:size])

        // Ukur waktu untuk Bubble Sort
        startBubble := time.Now()
        BubbleSort(sampleData)
        elapsedBubble := time.Since(startBubble)
        bubbleTimes = append(bubbleTimes, elapsedBubble.Seconds())

        // Ukur waktu untuk Recursive Insertion Sort
        startInsertion := time.Now()
        RecursiveInsertionSort(sampleData, 1)
        elapsedInsertion := time.Since(startInsertion)
        insertionTimes = append(insertionTimes, elapsedInsertion.Seconds())
    }
}
